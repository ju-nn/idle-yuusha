# 勇者は働きたくない。 最新プロトタイプ

テキストベースの放置系RPGブラウザゲームの現行プロトタイプです。
Melvor Idle 風に左メニューから仕事を選びつつ、ぼやきノベルを進行の中心に置いています。

世界観は「働きたくない勇者が、生活を整えながら少しずつ自由に近づく放置RPG」です。
魔王討伐よりも、仕事・農業・素材売却・戦闘・生活改善を通じて、今日の暮らしが少し楽になることを重視します。
詳細な世界観とぼやき文体ルールは `AGENTS.md` を参照してください。

## 起動

```bash
npm install
npm run dev
```

## 現在入っている主な仕様

- ぼやきノベル
  - 自動更新ではなく「次へ」で読む
  - `event` / `firstTime` / `failure` / `milestone` / `state` / `idle` / `daily` の7分類
  - 仕事Lvアップ、新エリア、新モンスター、アップグレード、状態、時間経過などに紐づいて発生する
  - 報酬は出来事が起きた時点で入り、ぼやきは未読としてあとから読める
- 左メニュー式の仕事選択
- 未解禁の仕事は `？？？`
- 仕事レベルでエリアや仕事が解禁
- 釣り / 休憩 / 木こり / 商人 / 農業 / 採掘 / 戦闘 / 王国勤務 / 在宅作業 / 高報酬依頼
- 商人
  - 放置仕事
  - 素材を自動売却してGに変換
- 戦闘
  - 通常放置仕事とは別UI
  - 戦闘ログあり
  - HP/MPを消費
- HP/MP
  - 戦闘用スタミナ
  - 時間で回復
- 農業
  - 植える→時間経過→収穫
  - 進行度バーあり
- 拠点整備
  - 旧「生活改善」
  - お金と素材を使ってHP/MP上限を上げる
- 図鑑
  - 発見物
  - ぼやき図鑑
  - 実績
- 実績
  - ユーモア重視の名前と説明
- ヘッダー
  - スクロールでコンパクト化
  - G / HP / MP / 魚 / 木 / 鉱のみ表示
  - 発見数とぼやき数は図鑑ボタンに統合
